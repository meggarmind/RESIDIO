# Condo Control Login Page - Detailed Description for Recreation

## Overall Layout
Split-screen design with two main sections side by side:
- **Left panel (~35% width)**: Login form on white/light gray background
- **Right panel (~65% width)**: Marketing hero section on dark teal background

---

## Left Panel - Login Form

### Header
- **Logo**: Teal/dark cyan house/building icon (two overlapping rectangular shapes suggesting a building)
- **Brand name**: "Condo Control" in dark gray/black text, clean sans-serif font
- Logo and text are horizontally aligned

### Form Container
- White card/panel with subtle border or shadow
- Generous padding around content (approximately 24-32px)
- Rounded corners (subtle, ~4-8px)

### Form Elements (top to bottom)

1. **Email Address field**
   - Label: "Email Address" (small, gray text positioned above input)
   - Input: Full-width text field with light gray border, white background
   - Standard height (~40-44px)

2. **Password field**
   - Label: "Password" (small, gray text positioned above input)
   - Input: Full-width text field with light gray border, white background
   - Standard height (~40-44px)

3. **Checkbox**
   - Unchecked square checkbox
   - Label: "Keep me logged in on this computer"
   - Text wraps to two lines
   - Checkbox aligned to top of text

4. **Login button**
   - Full-width button
   - Solid teal/dark cyan background
   - White text: "Login"
   - Rounded corners (approximately 4-6px border-radius)
   - Height: ~44-48px

5. **Forgot password link**
   - Centered text link: "I forgot my password"
   - Teal/cyan colored text with underline
   - Clickable link style

6. **Register button**
   - Full-width button
   - Same teal color as Login button
   - White text: "Register"
   - Rounded corners matching Login button
   - Height: ~44-48px

---

## Right Panel - Hero/Marketing Section

### Background
- Solid dark teal color (deep, rich teal approximately `#1a3a3a` or `#0d2f2f`)

### Content Layout
- Vertically centered content
- Horizontally centered
- Generous padding on all sides

### Content Elements (top to bottom)

1. **Main Heading**
   - Text: "Manage Community Assets"
   - Large white text (~36-48px)
   - Light/thin font weight (300-400)
   - Subtle underline decoration below text

2. **Subheading**
   - Text: "Accelerate the process of issuing vehicle permits, leasing bike racks & more"
   - Smaller white/light gray text (~16-18px)
   - Positioned directly below main heading
   - Regular font weight

3. **CTA Button**
   - Text: "Learn More"
   - Teal/cyan background (slightly lighter than page background, ~`#2d6a6a`)
   - White text
   - Pill-shaped/rounded corners (~20-24px border-radius)
   - Padding: ~12px vertical, ~24px horizontal

4. **Product Screenshot**
   - Laptop mockup displaying the application dashboard
   - Screenshot shows:
     - Left sidebar navigation with menu items
     - Main content area with "Request Community Asset" form
     - Form has stepper/progress indicator at top
     - Dropdown menus and form fields visible
   - Laptop shown at slight angle/perspective view
   - Drop shadow beneath laptop for depth

5. **Bottom Tagline**
   - Text: "Pay and reserve community assets with ease, whether it's bike racks, vehicle permits, tennis rackets, or garden plots."
   - White text, centered
   - ~14-16px font size
   - Positioned below the laptop mockup

## Color Palette

| Element | Color (Hex) | Description |
|---------|-------------|-------------|
| Primary teal (buttons) | `#2d6a6a` | Used for login/register buttons |
| Dark teal (right panel bg) | `#1a3a3a` or `#0d2f2f` | Hero section background |
| CTA button | `#3d7a7a` | Slightly lighter teal |
| White | `#ffffff` | Text on dark bg, form bg |
| Dark text | `#333333` | Form labels, brand name |
| Input border | `#cccccc` or `#d0d0d0` | Light gray borders |
| Link text | `#2d6a6a` | Forgot password link |
| Light gray bg | `#f5f5f5` or `#fafafa` | Left panel background |

---

## Typography

- **Font family**: Clean sans-serif (likely system font stack, Inter, Roboto, or Open Sans)
- **Hero heading**: 36-48px, font-weight 300-400 (light)
- **Hero subheading**: 16-18px, font-weight 400
- **Brand name**: 24-28px, font-weight 600
- **Form labels**: 12-14px, font-weight 400, color gray
- **Button text**: 14-16px, font-weight 500-600
- **Body/tagline text**: 14-16px, font-weight 400

---

## Spacing Guidelines

- **Left panel padding**: 32-48px
- **Form field gap**: 16-20px between fields
- **Button gap**: 12-16px between buttons
- **Right panel padding**: 40-60px
- **Content gap (right panel)**: 20-32px between elements

---

## Responsive Considerations

- On mobile, panels should stack vertically (login form on top, hero below or hidden)
- Form should remain functional at all breakpoints
- Laptop mockup may be hidden or scaled down on smaller screens

---

## Interactive States (Suggested)

- **Buttons**: Slightly darker on hover, subtle transition
- **Inputs**: Border color change on focus (teal highlight)
- **Links**: Darker shade on hover
- **Checkbox**: Teal checkmark when checked

---

## Logo Description (for recreation)

The logo consists of two overlapping rectangular shapes:
- Resembles an abstract building or house icon
- Two parallelogram/rectangle shapes offset diagonally
- Teal/cyan color matching the brand palette
- Simple, modern, geometric design
